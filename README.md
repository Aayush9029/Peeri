> [!NOTE]
> **Peeri** is currently in the design phase and not yet ready for public release. Contributions are welcome.

Peeri is a futuristic & cutting-edge, native macOS torrent client designed for the modern era. It features system-level integration with macOS, making it a great choice for users in the Apple ecosystem.
